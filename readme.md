# Github keyboard shortcuts extension

This firefox browser extension provides keyboard shortcuts for navigating one of the poorest user interface known to mankind - github.com. It's currently WIP and offers hard-coded opinionated shortcuts for navigating repositories, pull requests and issues.

## Available shortcuts

- `c`, `i`, `p` and `a` - navigate through repository tabs
- `o` - jump to the repository's owner (organization or user) page
- `r` - jump to the README on the repository root
- `j` / `↓` and `k` / `↑` - move the selection through the pull request list (on the `pulls` tab)
- `Enter` - open the currently selected pull request
